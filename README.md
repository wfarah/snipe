# SNIPE -- Signal-to-Noise Investigation of Pulsed Events

 This graphical interface allows users to:
 - Load a filterbank file
 - Dedisperse the data
 - Manually remove RFI
 - Average in time and frequency
 - Calculate SNR using selected signal and noise regions
